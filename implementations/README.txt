* Code was tested on the following system:
- 12 GB RAM
- matlab 2018a
- Red Hat Enterprise Linux 7.4 

* How to run on linux terminal: 
- run the "run_sim.sh" on the command line.

** Important note **
- To run on command line using the "run_sim.sh": Make sure to change the path in the driver.m file to be the path of the code on your system.
- To run inside Matlab: Make sure to add the "molecular_simulations" folder to the Path.
